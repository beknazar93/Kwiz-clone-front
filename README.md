
# Kwiz Clone Frontend

Фронтенд-клонирование приложения Kwiz, разработанное с использованием современных технологий.

## ⚙️ Стек технологий

- React
- React Router
- SCSS / CSS Modules
- Axios
- Vite

## 🚀 Начало работы

2. Установка зависимостей
Перейдите в директорию проекта и установите необходимые пакеты:

bash
Копировать
Редактировать
cd Kwiz-clone-front
npm install
3. Запуск проекта в режиме разработки

Копировать
Редактировать
